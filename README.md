# mhat-visualstudio-template

為部落格系列：[打造自己的template-建立一致性程式碼](http://blog.alantsai.net/search/label/%E3%80%8C%E6%89%93%E9%80%A0%E8%87%AA%E5%B7%B1%E7%9A%84template-%E5%BB%BA%E7%AB%8B%E4%B8%80%E8%87%B4%E6%80%A7%E7%A8%8B%E5%BC%8F%E7%A2%BC%E3%80%8D) 裡面的sample程式碼用來demo如何建立Visual Studio Template。

## 快速使用 (getting started)

可以透過 git clone的方式把此專案clone下來，透過切換tag來看某一個章節相關的程式碼範例。

下面例子是
1. clone專案下來
2. cd進去專案
3. 切換顯示第2篇的相關sample程式

```
git clone https://github.com/alantsai-samples/mhat-visualstudio-template.git
cd mhat-visualstudio-template
git checkout samplte/chapter02
```

如果只想要看到最新的程式碼內容就好

可以直接下載最新版本的zip檔案：

[下載最新的zip](https://github.com/alantsai-samples/mhat-visualstudio-template/archive/master.zip)
